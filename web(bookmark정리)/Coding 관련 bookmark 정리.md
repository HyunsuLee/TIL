## 모니터 먹통 현상 해결
### input source 바꾸기 등으로 우분투로 부터의 display가 안뜰때
ctrl + alt + F1 으로 가상터미널로 들어가 display가 뜨는걸 확인하고(~~optional~~ display output을 refresh 필수인듯)

ctrl + alt + F7 으로 GUI환경으로 다시 돌아 온다.
